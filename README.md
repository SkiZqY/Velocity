# Velocity Launcher
为电竞而生，打造最完美的电子竞技体验！功能强大的游戏启动器，提升您的游戏体验


## 项目简介

Velocity Launcher 是一款专为游戏玩家设计的启动器，提供游戏优化、进程管理、显卡设置和部署功能，帮助玩家获得更好的游戏性能和稳定性。

## 主要功能

- **游戏优化**：自动调整系统设置，提升游戏性能
- **进程管理**：监控和管理系统进程，释放系统资源
- **显卡设置**：支持修改显卡驱动名称，适用于特定游戏场景
- **ICC 管理**：管理色彩配置文件，提升游戏视觉体验
- **部署功能**：快速部署和安装游戏，管理更新
- **个人资料**：查看和管理个人信息，集成 Steam 账号

## 系统要求

- Windows 10/11 64位
- Intel Core i5 或 AMD Ryzen 5 以上
- 8GB 以上内存
- NVIDIA GeForce GTX 1060 或 AMD Radeon RX 580 以上
- 至少 500MB 可用空间

## 安装

1. 克隆或下载本仓库
2. 运行安装程序，按照提示完成安装
3. 启动 Velocity Launcher 开始使用

## 使用方法

### 游戏优化
1. 在左侧菜单中选择"优化"
2. 点击"一键优化"按钮，自动优化系统设置

### 进程管理
1. 在左侧菜单中选择"调度"
2. 查看当前运行的进程
3. 选择不需要的进程，点击"结束进程"按钮

### 显卡设置
1. 在左侧菜单中选择"显卡"
2. 选择目标显卡型号或自定义型号
3. 点击"执行修改"按钮
4. 重启电脑或驱动使设置生效

### ICC 管理
1. 在左侧菜单中选择"ICC"
2. 加载或创建色彩配置文件
3. 应用配置文件到游戏

### 部署功能
1. 在左侧菜单中选择"部署"
2. 选择游戏安装包
3. 点击"开始部署"按钮

### 个人资料
1. 在左侧菜单中选择"资料"
2. 查看个人信息和 Steam 账号信息
